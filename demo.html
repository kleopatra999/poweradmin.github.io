Demo

Get a feel for the interface

The demo will allow you to get a feel for the interface. The interface is normally running the latest code.

You can login as a user using the username and password "demo". There is no PowerDNS instance running against this database, it is just the Poweradmin interface and a mysql backend. The database is regularly emptied. Please be respectful to other users checking out the demo - so don't change the password of the demo user, for example (and if you do, it will be reset within a couple of hours anyway).
